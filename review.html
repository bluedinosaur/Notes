GIT & GITHUB

Name Tag Example Description
Upload new project to Github //To upload a new project...
  1. cd over to project directory
  2. go to github, and create new project (with readme)
  3. copy the http link
  4. in the terminal, git clone (http link). This clones the github repository to your computer
  5. cd over to cloned folder, and check if readme and hidden git file is there
  6. git status to check the untracked files
  7. git add files you want to upload/sync to github repo
  8. git status to check
  9. git commit -m "message note" to commit the changes to github with a message
  10. git push to upload/sync changes to github
  11. git pull to update local files from the version on github
additional notes...
  • git add -A stages all changes
  • git add . stages new files and modifications, without deletions
  • git add -u stages modifications and deletions, without new files
Branch from github //Create a new branch from the master...
  1. git branch to check what branch you are currently on
  2. git branch newfeature to create a new branch called newfeature
  3. git branch to check the created branch
  4. git checkout newfeature to switch to the branch "newfeature"
