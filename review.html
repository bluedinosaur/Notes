Pass by REFERENCE
(shallow object cloning) 		//EXAMPLE5 - To clone an object...
let obj = {a: "a", b: "b", c: {text: "Hello"}}

Example 5 - This example of shallow cloning where we can only clone the first layer. This can cause problems because the second layer across all variables can be affected by changes even though it is cloned.
Pass by REFERENCE
(deep object cloning) 		//EXAMPLE6 - To clone an object...
let obj = {a: "a", b: "b", c: {text: "Hello"}}

let objClone1 = Object.assign({}, obj);
let objClone2 = {...obj}
let deepClone = JSON.parse(JSON.stringify(obj))
obj.c.text = "Goodbye";

console.log(obj);
console.log(objClone1);
console.log(objClone2);
console.log(deepClone);

//{a: "a", b: "b", c: {text: "Goodbye"}}
//{a: "a", b: "b", c: {text: "Goodbye"}}
//{a: "a", b: "b", c: {text: "Goodbye"}}
//{a: "a", b: "b", c: {text: "Hello"}}
JSON.stringify makes the object into a string, then is parsed back into an object. This makes it unaffected by the shallow cloning.
